# musenginetest
 attempt at a sequenced music playback engine.
 plays music from .csv files.
 loads samples from raw files.
 Controls:
  F1: numbers view
  F2: piano view
  0-9: mute/unmute channels
  up/down: increase/decrease pitch
  left/right: speed up/down
  space: pause
  r: restart song
  b: load another song
  q: quit
